# Manifeste Octopus Engine

Octopus Engine existe pour une raison simple : chaque nouvelle application IA recrée trop souvent les mêmes briques.

Mémoire, conversation, connecteurs, planification, génération, publication, analyse, surveillance, permissions : à chaque projet, on recommence.

Octopus propose une autre voie.

Construire une seule base capable d'apprendre plusieurs métiers, sans imposer une seule personnalité ni une seule application.

## Devise

> Une intelligence n'a pas vocation à montrer sa complexité. Elle a vocation à simplifier celle des autres.

## Pourquoi Octopus existe

Octopus n'existe pas pour remplacer les humains.

Il n'existe pas pour remplacer les applications.

Il existe pour absorber la complexité afin que chaque application puisse rester simple.

L'utilisateur garde l'intention. Le moteur absorbe la mécanique.

## Ce que nous refusons

Nous refusons :

- le gros prompt central qui sait tout ;
- les modules qui se parlent directement ;
- les connecteurs qui contiennent de la logique métier ;
- les fonctionnalités ajoutées sans besoin réel ;
- les métaphores poétiques transformées en dette technique ;
- l'IA qui improvise quand elle devrait reconnaître ses limites.

## Ce que nous construisons

Nous construisons un moteur :

- composable ;
- observable ;
- sobre ;
- cloisonné ;
- versionné ;
- gouverné par décisions explicites ;
- capable d'évoluer sans perdre son identité.

## Le langage du poulpe

La métaphore du poulpe sert à penser le système : un centre qui orchestre, des bras spécialisés, une communication minimale, une intelligence distribuée mais une expérience utilisateur cohérente.

Cette métaphore ne doit jamais dicter directement l'implémentation.

Dans la Bible, nous pouvons parler de tentacules, de nid, de mue et de digestion.

Dans le code, nous parlons de modules, de runtime, de policies, de workflows, d'états et de contrats.

## La promesse

Octopus Engine n'est pas conçu pour devenir le plus gros moteur d'IA.

Il est conçu pour rester le plus élégant possible, quel que soit son âge.
