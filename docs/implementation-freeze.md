# Implementation Freeze

Octopus Engine reste en phase documentation tant que les contrats minimaux ne sont pas définis.

## Gel temporaire

Pas d'implémentation runtime avant :

- schéma Workflow ;
- schéma Module Task ;
- schéma Capability ;
- schéma Policy ;
- schéma Trace ;
- tests de frontières.

## Pourquoi

Le but est d'éviter qu'une première implémentation accidentelle devienne l'architecture réelle.

Le poulpe doit savoir où sont ses bras avant de remuer la mer.
