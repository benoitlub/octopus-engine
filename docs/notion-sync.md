# Notion ↔ GitHub

## Principe

GitHub est la source technique durable.

Notion est la Bible vivante, l'espace de synthèse et de lecture humaine.

## Règle

Les décisions structurantes doivent exister dans GitHub sous forme de fichiers versionnés.

Notion peut les résumer, les relier, les expliquer et les rendre lisibles.

## Source de vérité

- Constitution : GitHub prioritaire.
- ADR : GitHub prioritaire.
- Bible : Notion et GitHub doivent rester alignés.
- Notes exploratoires : Notion prioritaire au début, puis GitHub quand une décision devient stable.

## Synchronisation manuelle

Pour l'instant, la synchronisation est manuelle.

Un futur connecteur Notion pourra lire ou résumer la Bible, mais il ne doit pas modifier les documents fondateurs sans validation humaine.
