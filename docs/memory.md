# Memory

Memory désigne le mécanisme de mémoire commun.

Les espaces mémoire restent cloisonnés par application, tenant et contexte.

## Rôle

- retenir ;
- classer ;
- relier ;
- résumer ;
- oublier ;
- fournir du contexte autorisé.

## Non-rôle

Memory ne décide pas de la mission.

Memory ne lit pas les espaces d'autres applications sans autorisation explicite.

Memory ne contourne jamais Gateway, Policies ou Guardian.

## Digestion

Dans le langage humain, Memory est l'estomac d'Octopus.

Dans le langage technique, c'est un mécanisme de stockage, récupération, filtrage et rétention de contexte.

## À définir

- schéma de mémoire ;
- règles de rétention ;
- droit à l'oubli ;
- niveaux de confidentialité ;
- tracing des accès mémoire.
