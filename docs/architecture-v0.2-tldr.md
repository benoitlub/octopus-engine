# TL;DR Architecture v0.2

Octopus Engine est un moteur agnostique.

Les applications apportent la personnalité.

Le Conductor parle à l'utilisateur.

Le Coordinator exécute les workflows.

Le Workflow est une partition déclarative.

Chaque module reçoit seulement sa tâche locale.

Les Capabilities sont les compétences fines.

Les Connectors touchent le monde extérieur.

Guardian protège le runtime sans décider du métier.

La biologie du poulpe sert à penser, pas à coder.

Documentation d'abord. Code ensuite.
