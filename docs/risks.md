# Risques connus

## Coordinateur qui grossit

Risque : le Coordinateur accumule la connaissance métier.

Réponse : workflows déclaratifs comme objets de première classe.

## Guardian qui devient juge

Risque : Guardian prend des décisions métier sous prétexte de sécurité.

Réponse : Loi XIV et ADR-0005.

## Métaphore qui contamine le code

Risque : interfaces anthropomorphiques inutiles.

Réponse : Loi XIII et ADR-0003.

## Modules qui se connaissent

Risque : couplage direct entre modules.

Réponse : tâches locales, tests de frontières.

## Documentation qui diverge

Risque : Notion et GitHub racontent deux histoires différentes.

Réponse : GitHub source technique durable, Notion Bible vivante alignée.

## Exemple qui devient spec

Risque : un exemple non validé devient contrat par copie-collage.

Réponse : avertissement explicite et ADR obligatoires.
