# Avertissement sur les exemples

Les exemples présents dans ce dépôt sont conceptuels tant qu'une ADR n'a pas défini un schéma stable.

Un exemple ne doit jamais devenir une spécification par accident.

## Règle

- Exemple = aide à comprendre.
- ADR = décision.
- Spécification = contrat.
- Code = implémentation.

Si un exemple est copié dans du code avant validation, le poulpe mord le câble.
