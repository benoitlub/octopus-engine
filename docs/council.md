# Conseil des Céphalopodes

Le Conseil des Céphalopodes est le rituel de gouvernance d'Octopus Engine.

## Rôles actuels

- Benoît : vision, arbitrage, produit.
- ChatGPT : synthèse, cohérence, documentation.
- Claude : critique architecturale et philosophie.
- Gemini : rigueur technique, état de l'art, scalabilité.
- Codex : implémentation.

## Règle de décision

Les avis se confrontent.

Les décisions sont arbitrées par le gardien de la vision.

Les personnes sont respectées.
